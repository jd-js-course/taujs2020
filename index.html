Javascript for the Arts

Tel Aviv University

Introduction

Nowadays, more than ever, computers are a vital part of the art world. This course focused on presenting software development skills to art students.

During the course, the students learned to realize artistic ideas using the Javascript computer language, emphasizing visual and interactive applications. Finally, the students were introduced to various software engineering concepts and ideas while reviewing software art projects.

This website presents the students' final projects. Every project had to include visual and sound elements while being interactive and artistic. But most importantly, it had to be fun.

All the source code is available in the course's repository.
